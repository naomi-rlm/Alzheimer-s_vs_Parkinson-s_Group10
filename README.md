# Alzheimer-s_vs_Parkinson-s_Group10
### Naomi Le Mouel

## Code Folder:
### Alz_Pd_DataCleaning:
Notebook contains data cleaning for the GWAS dataset as well as initail EDA for this data.
### Alz_PD_Modeling:
Notebook contains work to extract ClinVar data VIA API and data cleaning/schema standardizing to match the column names and structure of the GWAS dataset. Modeling work occurs after these steps.

## Data Folder:
### Alz_Pd_Clean:
This is the cleaned and integrated GWAS dataset.
### MONDO_0004975_Alzheimers:
This is the raw GWAS Alzeheimer's dataset.
### MONDO_0005180_Parkinsons:
This is the raw GWAS Parkinson's dataset
